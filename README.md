# 数独ゲーム - Minimal Hints Challenge

ブラウザで遊べる数独ゲームです。最小ヒント数の難しい問題にも挑戦できます。

---

## 特徴
- **一意解保証**: 必ず一意の解を持つ数独を自動生成  
- **最小ヒント数**: Python版ロジックを参考にした、一意解を保つ最小ヒント数問題  
- **ブルータリストデザイン**: モノクロ・角ばったスタイル・Courier Newフォント  
- **レスポンシブUI**: スマートフォンでも快適にプレイ可能  
- **タイマー機能**: プレイ時間を記録  
- **メモ機能**: 候補数字をセル内に小さく表示可能  
- **EXTREMEモード**: 7種類の模様（ハート・星・十字など）を持つ特別配置  

---

## 遊び方
1. 「新規生成」または「EXTREME生成」で問題を作成  
2. マスをクリックして選択し、数字ボタンまたはキーボード（1〜9）で入力  
3. 「正誤確認」で間違いをチェック（赤色表示）  
4. 「メモモード」をONにすると候補数字を小さく記入可能（`M`キーで切り替え）  
5. 「解答表示」で完全な解答を確認可能  

---

## EXTREMEモードの種類
7種類のパターンをランダム選択：
1. **ハート**: 外側から中心に向かって削除するハート形  
2. **星**: 6つの頂点から中心へ伸びる配置  
3. **対角線**: 2本の斜めライン  
4. **十字**: 縦横の十字配置  
5. **左上**: 左上に集中した配置  
6. **額縁**: 同心正方形  
7. **市松模様**: チェッカーボード風配置

## ローカルで実行
```bash
# このリポジトリをクローン
git clone https://github.com/meratory5/sudoku_web.git

# ディレクトリに移動
cd sudoku_web

# index.htmlをブラウザで開く（Windows）
start index.html

# macOS / Linux
open index.html
```
または `index.html` を直接ダブルクリックでも起動できます。

---

## GitHub Pagesでの公開
1. GitHubにリポジトリを作成  
2. このコードをプッシュ  
3. 「Settings」→「Pages」→ “Deploy from a branch” を選択  
4. mainブランチを指定して保存  
5. 数分後に  
   `https://YOUR-USERNAME.github.io/sudoku_web/`  
   でアクセス可能になります。
